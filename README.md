# 📊 Retail Sales & Customer Insights Dashboard

## 📌 Project Overview

This project analyzes retail customer shopping behavior using SQL, PostgreSQL, and Power BI. The objective was to transform raw customer transaction data into meaningful business insights through an interactive dashboard.

---

## 🛠️ Tools Used

- PostgreSQL
- SQL
- Power BI

---

## 📂 Dataset

- Customer Shopping Behavior Dataset
- 3,900 Records
- 18 Columns
